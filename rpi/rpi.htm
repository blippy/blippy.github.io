Raspberry Pi

Boot to sata

sudo nano /boot/cmdline.txt
so that
root=/dev/sda2

Creating a root image on a USB

Somewhat incomplete, but should get you much of the way there. 
sudo mkfs -t ext4 /dev/sda1 # prolly unnecesary
sudo dd if=/dev/root of=/dev/sda1 bs=4M
sudo e2fsck /dev/sda1
sudo resize2fs /dev/sda1

Power usage

According to Yahoo Answers, USB pen drives consume about 5v @ 145mA (725mW) at their heaviest load, in write mode. On 15-Sep-2013, I estimated that the yearly power consumption of an RPi was about £5 (DB04/89).

Wifi (Pi2)

Assumes use of Edimaxx Wifi dongle. Pi2 keeps turning off the wifi, so to disable it: 
	sudo vim /etc/network/interfaces
Add the line:
	wireless-power off
