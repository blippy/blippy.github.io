fstab

Identify what devide it is you need to add:
  sudo blkid

Add it to /etc/fstab:
  /dev/sda2 /mnt/win7 ntfs-3g defaults,uid=3003,rw 0 0
  /dev/sdb1 /mnt/sdb1 ntfs defaults,user,rw 0 0

Mount and unmount:
  sudo mount /dev/sdb1
  sudo umount /dev/sdb1
