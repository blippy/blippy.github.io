Aerosnap

In ~/.fluxbox/keys: 
# Tiling

#Control Mod1 Left :MacroCmd {ResizeTo 50% 100%} {MoveTo 00 00 Left}

Mod4 Left :MacroCmd {ResizeTo 50% 100%} {MoveTo 00 00 Left}
Mod4 Right :MacroCmd {ResizeTo 50% 100%} {MoveTo 00 00 Right}
Mod4 Up :MacroCmd {ResizeTo 100% 50%} {MoveTo 00 00 Up}
Mod4 Down :MacroCmd {ResizeTo 100% 50%} {MoveTo 00 00 Bottom}
Mod4 Return :ToggleCmd {Maximize} {Restore}

# Useful shortcuts
Mod4 f :exec firefox
Mod4 m :exec thunderbird
Mod4 t :exec xterm

`Mod4 ' is the left-window button. These keys may already by used, e.g. by :SendToPrevWorkspace, so you may have to comment out the latter.

Background colour

In ~/.fluxbox/startup, add the following line before exec fluxbox: 
/usr/bin/fbsetroot -solid gray30

Start menu

In $HOME/.fluxbox/init, you need to add RootMenu: 
session.screen0.toolbar.tools:  RootMenu, iconbar, systemtray, clock
To prevent user from dragging window into another workspace: 
session.screen0.workspacewarping:      false

Time

Change the line in ~/.fluxbox/init to: 
session.screen0.strftimeFormat: %a %x %X
