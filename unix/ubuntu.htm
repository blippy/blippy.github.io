Ubuntu

Some set-up notes. Geared towards 18.04.

cpufreq

List the available governors:
	cpufreq-info -g
List the governors being used by cpu:	
	cat /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor
Set the governor for all cpus:
	sudo cpufreq-set -r -g powersave

Create launcher item

The conventional location is ~/.local/share/applications, but it will still work if you keep it in a place like /home/mcarter/redact/docs/ernst-desktop.

Create a file called ernst.desktop: 

[Desktop Entry]
Name=ernst
Comment=mcarter
Exec=ssh -p 2222 -X mcarter@ernst
Icon=/home/mcarter/redact/docs/ernst-desktop/ernst.png
Terminal=true
Type=Application
StartupNotify=true
Drag the resulting file onto the launcher.

Deactivate on-line search

Turn off on-line search results, which should speed things up:System Settings: Security & Privacy: Search: switch off Include online search results

Gnome

sudo apt install dconf-editor # graphical settings

Customise the clock:
	org.gnome.desktop.interface.clock-show-date

Enable	Ctl-Alt-Bspace:
	gsettings set org.gnome.desktop.input-sources xkb-options "['terminate:ctrl_alt_bksp']"
