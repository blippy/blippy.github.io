Profiling

gprof

gprof as been used effectively in project neoleo via the macro m4/ax_gprof.m4, which I wrote myself.

Compile using debugging (e.g. -O0 -g) and -pg flag: 

g++ foo.cc -pg -O0 -g -o foo

Run the executable to create the binary profile file gmon.out: 

foo

Create a human-readable report: 

gprof foo gmon.out > report.txt
