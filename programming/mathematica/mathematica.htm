Financial data

FinancialData[] seems to no longer work (as at Nov 2017).

Exploring financials: 

x = Entity["Financial", "L:VOD"]
x["Properties"]
EntityValue[x, "Close"]

Import table: 

tbl1 = Import["/home/pi/shares.tsv"];
price[sym_] := EntityValue[Entity["Financial", "L:" <> sym, "Open"]
f[row_] := (r1 = row[[1]]; r2 = row[[2]]; sp = price[r1]; {r1, r2, sp, r2+sp})
tover[fn_] := Table[fn[tbl1[[r]]], {r, Length[tbl1]}]
tbl2 = tover[f]
tbl3 = Grid[tbl2, Alignment -> Right]
Total[tbl2[All, 4]]] + Quantity[123.45 + 456.78, "GBP"]
