guile

Init file

Typically in ~/.guile: 
(use-modules (ice-9 readline))
(activate-readline)

Modules

E.g. 
(use-modules (srfi srfi-13))

Parsing

Using the lalr parser: 
(use-modules (system base lalr))
Starting from version 2.2, guile has a PEG parser.
