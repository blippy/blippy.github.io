FidoCadj

A good intuitive tool (xlink). To use: 
java -jar fidocadj.jar

Fritzing

Install: 
apt install fritzing fritzing-parts
Run: Fritzing (note caps).

xcircuit

It works, but seems a bit finicky, and crashed. Use FidoCadj instead.

Powering

Barrel: 9-12V DC, minimum 250mA, preferrably 500mA or even 1A for heavy circuits. Plug must be centre-positive.

Setup

Tools : Port : /dev/ttyACM0

sudo usermod -a -G dialout,tty USER
sudo chmod 666 /dev/ttyACM0
sudo chmod a+rw /dev/ttyUSB0
Icon:
  1. run install.sh. This creates a desktop item in ~/Desktop
  2. Double-click on it
  3. In the Ubuntu icon bar, add icon to favourite
  4. Close the IDE
  5. Remove the file in ~/Desktop
