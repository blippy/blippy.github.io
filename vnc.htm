VNC

Clients

Remmina and TigerVNC Viewer seem to work acceptably.

Debian 10 server

Edit ~/.vnc/xstartup: 
#!/bin/env bash
exec /usr/bin/mate-session &
Start vnc: 
	vncserver :1  -geometry 1920x1080 -localhost no
or
	vncserver  -geometry 1920x1080 -localhost no
or
	vnc -c # my own script

Raspberry Pi OS server

MATE on Buster didn't seem to start properly.

LXDE seems the best bet. You need to install it first, though. Edit ~/.vnc/xstartup: 

#!/bin/env bash
exec startldxe &

Don't use the default VNC server. Xtigervnc seems a good bet.
