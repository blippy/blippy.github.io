Github, and alternatives.

Github

  • Clone: git clone git@github.com:blippy/tacc.git
  • Pushing back: commit your changes, then: git push. If you get an error message like You can't push to git://github.com/user/repo.git, then you need to edit .git/config so that the url line reads: url = git@github.com:blippy/tacc.git

    Forks

    Merging from upstream: 

      git fetch upstream
  git merge upstream/master
  git push # push the merge to your github fork
    xlink Also explains how to set "upstream".

    Upstream

    Add grandparent to upstream:
    git remote add upstream https://github.com/sean-lawless/computersystems.git
    git remote -v # List the remotes

To create a branch, e.g. foo, based on grandparent's master:
    git checkout -b foo upstream/master
