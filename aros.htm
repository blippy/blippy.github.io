FTP

The best way to save data is via FTP using client software on AROS, and server software on Linux.

Client-side

On AROS, use the client: AROS: Tools: MarranoFTP

Server-side

You can use a proper FTP server, but that's usually too much effort to set up. I prefer to use the python 2/3 FTP library pyftpdlib, which you can install using a command like: 
sudo pip install pyftpdlib
You will need to create the server software. I have created a gist for that purpose. Place it in the directory you want to serve from, and start it via: 
python3 pyftpd.py
This will create an FTP server with username user and password 12345 on port 2121. You should use these settings in MarranoFTP.

Security alert: there is no security! Make sure you run the server behind a firewalled router so that you are not exposing you data over the whole of the Interwebs. Close the sever when you no longer need it.
